# Webchat
Golang 으로 만들어 보는 websocket 을 이용한 공개 채팅 애플리케이션

## Getting started

### Prerequisites

* Prepared MacBook ;)

* [Install Homebrew](https://brew.sh)

* Install Golang
```shell
$ brew install go
```

* Install Node.js
```shell
$ brew install node
```

### Mac OSX
* Get source code
```shell
$ git clone git@github.com:loganstone/webchat.git
```

* Assets install
```shell
$ cd webchat;npm install
```

* Code build
```shell
$ go build
```

* Server run
```shell
$ ./webchat
```

### First contact
Click! http://localhost:8080
